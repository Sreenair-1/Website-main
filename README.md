# Project Overview

## Features
- User authentication
- Responsive design
- RESTful API integration
- Cross-browser compatibility

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Rancidcake/Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
